# 🚀 Personal Portfolio

A modern and fully responsive Full Stack Developer Portfolio built with **React.js**, **Vite**, **Node.js**, **Express.js**, and **MongoDB**. This portfolio showcases my projects, technical skills, and provides a fully functional contact form connected to a backend and database.

🌐 **Live Demo:**  
👉 https://personal-portfolio-jatin.vercel.app/

## ✨ Features

- 🎨 Modern Glassmorphism UI
- 🌙 Dark Theme
- 📱 Fully Responsive Design
- ⚡ Smooth Animations using Framer Motion
- 👨‍💻 Project Showcase
- 🎥 Video Preview for Projects
- 📄 Resume Download
- 📬 Contact Form with Backend Integration
- 💾 MongoDB Database Storage
- 🚀 Deployed using Vercel & Render

## 🚀 Getting Started

### Clone Repository

```bash
git clone https://github.com/Jeetvk18/personal_portfolio_jatin.git
```

---

### Frontend

```bash
cd portfolio
npm install
npm run dev
```

---

### Backend

```bash
cd server
npm install
npm start
```

---

## ⚙ Environment Variables

Create a `.env` file inside the `server` folder.

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
JWT_SECRET=your_secret_key
```
